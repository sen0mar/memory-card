# A SIMPLE MEMORY CARD GAME

This is a simple memory card game Made to practice react useState and useEffect Hooks, mobile layout,
and accessibility.

Demo: https://memory-card-at6.pages.dev/

# INSTRUCTIONS

The game consists on being able to select a number of cards without choosing the same one twice in the same session.
Choose a difficulty to start the game: for easy mode, you will need to reach a score of 6. For medium, a score of 10, and for hard, a score of 20.

If you look at the instructions again, click the help icon on the top left. If you want to change difficulty mid-game,
click the Change Difficulty button on the top right.
